# Agent AI Tactical Strategic Example

This example demonstrates a simple agentic AI preparing a meal. It illustrates the difference between a 'tactical' approach, where the agent reacts to immediate needs, and a 'strategic' approach, where it follows a phased plan to ensure prerequisites are met comprehensively before proceeding.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal: `python main.py`
